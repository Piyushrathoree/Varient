## Installation

Copy the component from `packages/ui/src/components/foundation/sidebar/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import {
  Sidebar,
  SidebarFooter,
  SidebarHeader,
  SidebarNav,
  SidebarNavItem,
  SidebarSection,
  SidebarToggle,
  useSidebar,
} from '@/components/foundation/sidebar';
import { Home, Layers } from 'lucide-react';

function Brand() {
  const { isCollapsed } = useSidebar();
  if (isCollapsed) return null;
  return <span className="font-semibold">Acme</span>;
}

export function AppShell() {
  return (
    <div className="flex h-screen">
      <Sidebar defaultCollapsed={false}>
        <SidebarHeader>
          <Brand />
          <SidebarToggle />
        </SidebarHeader>
        <SidebarNav>
          <SidebarSection label="Main">
            <SidebarNavItem href="/" icon={<Home className="size-4" />} label="Home" isActive />
            <SidebarNavItem href="/layers" icon={<Layers className="size-4" />} label="Layers" badge="3" />
          </SidebarSection>
        </SidebarNav>
        <SidebarFooter>{/* footer items */}</SidebarFooter>
      </Sidebar>
      <main className="flex-1">{/* content */}</main>
    </div>
  );
}
```

## Anatomy

- `Sidebar` — root `<aside>` with animated width (256px expanded, 72px collapsed icon rail).
- `SidebarHeader` / `SidebarFooter` — top and bottom chrome slots.
- `SidebarNav` — `<nav>` region with `aria-label`.
- `SidebarSection` — labelled group (`role="group"`).
- `SidebarNavItem` — icon + label link with `aria-current="page"` when `isActive`.
- `SidebarToggle` — collapse/expand control.
- `useSidebar()` — read `isCollapsed`, `setCollapsed`, and `toggleCollapsed` from children.

## Accessibility

- Root renders `<aside aria-label="Sidebar">`.
- Navigation uses semantic `<nav>` with configurable `aria-label`.
- Active items set `aria-current="page"`.
- Collapsed items expose `title` tooltips with the label text.
- Width animation and label fade respect `prefers-reduced-motion`.
