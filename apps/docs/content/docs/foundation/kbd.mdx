## Installation

Copy the component from `packages/ui/src/components/foundation/kbd/` into your project.

```bash
bun add clsx tailwind-merge
```

## Usage

```tsx
import { Kbd, KbdGroup } from '@/components/foundation/kbd';

export function Example() {
  return (
    <p className="text-sm text-muted-foreground">
      Press <KbdGroup keys={['⌘', 'K']} size="sm" /> to search.
    </p>
  );
}
```

Single keys:

```tsx
<Kbd>Esc</Kbd>
```

## Accessibility

- Renders semantic `<kbd>` elements for individual keys.
- `KbdGroup` wraps keys in a `role="group"` container; + separators are `aria-hidden`.
- Use alongside visible instructions — shortcuts alone are not sufficient context for all users.
