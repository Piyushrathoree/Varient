## Installation

Copy the component from `packages/ui/src/components/foundation/separator/` into your project.

```bash
bun add @radix-ui/react-separator clsx tailwind-merge
```

## Usage

```tsx
import { Separator } from '@/components/foundation/separator';

export function Example() {
  return (
    <div className="flex flex-col gap-4">
      <p>Sign in with email</p>
      <Separator label="or continue with" />
      <p>Social providers</p>
    </div>
  );
}
```

Vertical divider:

```tsx
<div className="flex h-12 items-center gap-4">
  <span>Left</span>
  <Separator orientation="vertical" />
  <span>Right</span>
</div>
```

## Accessibility

- Built on `@radix-ui/react-separator` — sets `role="separator"` when not decorative.
- Defaults to `decorative={true}` for visual-only dividers; set `decorative={false}` when the separator conveys meaningful structure.
- Labelled horizontal variant uses `role="none"` on the wrapper so assistive tech focuses on content sections, not the divider text.
