## Installation

Copy the component from `packages/ui/src/components/foundation/progress/` into your project.

```bash
bun add @radix-ui/react-progress motion clsx tailwind-merge
```

## Usage

```tsx
import { Progress } from '@/components/foundation/progress';

export function Example() {
  return (
    <Progress value={65} max={100} showValueLabel aria-label="Upload progress" />
  );
}
```

For loading states where completion percentage is unknown:

```tsx
<Progress isIndeterminate aria-label="Loading" />
```

## Accessibility

- Built on `@radix-ui/react-progress` — exposes `role="progressbar"` with `aria-valuenow`, `aria-valuemin`, and `aria-valuemax` when determinate.
- Indeterminate mode omits a fixed value and shows an ellipsis in the optional label.
- Sweep animation is replaced by a static 50% bar when `prefers-reduced-motion` is enabled.
