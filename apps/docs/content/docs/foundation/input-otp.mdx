## Installation

Copy the component from `packages/ui/src/components/foundation/input-otp/` into your project.

```bash
bun add input-otp clsx tailwind-merge
```

## Usage

```tsx
import { useState } from 'react';
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSeparator,
  InputOTPSlot,
} from '@/components/foundation/input-otp';

export function Example() {
  const [code, setCode] = useState('');

  return (
    <InputOTP length={6} value={code} onChange={setCode}>
      <InputOTPGroup>
        <InputOTPSlot index={0} />
        <InputOTPSlot index={1} />
        <InputOTPSlot index={2} />
      </InputOTPGroup>
      <InputOTPSeparator />
      <InputOTPGroup>
        <InputOTPSlot index={3} />
        <InputOTPSlot index={4} />
        <InputOTPSlot index={5} />
      </InputOTPGroup>
    </InputOTP>
  );
}
```

## Accessibility

- Built on `input-otp` — a single hidden input drives all slots for screen readers and mobile OTP autofill (`autocomplete="one-time-code"`).
- Supports iOS/Android paste, cut, and SMS code retrieval.
- Active slot shows a brand focus ring; invalid state sets `aria-invalid` on slots.
- Caret blink uses `motion-reduce:animate-none` for reduced-motion users.
