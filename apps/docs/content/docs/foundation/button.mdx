## Installation

Copy the component from `packages/ui/src/components/foundation/button/` into your project.

```bash
bun add clsx tailwind-merge
```

## Usage

```tsx
import { Button } from '@/components/foundation/button';

export function Example() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  );
}
```

## Variants

- **primary** — brand gradient fill, main CTAs
- **default** — high-contrast neutral
- **secondary** — muted background
- **outline** — bordered, transparent
- **ghost** — minimal, hover fill
- **destructive** — danger actions
- **link** — text-only with underline on hover
- **sweep** — brand-outlined pill; on hover a solid fill glides in from the left on a GPU `scaleX` transform (not a box-shadow), so it stays smooth at any size
- **frame** — brand-gradient rimmed surface that lifts and grows its shadow on hover
