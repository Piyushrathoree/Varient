## Installation

Copy the component from `packages/ui/src/components/foundation/command-palette/` into your project.

```bash
bun add cmdk @radix-ui/react-dialog motion clsx tailwind-merge
```

## Usage

```tsx
import { useState } from 'react';
import { CommandPalette, Button, KbdGroup } from '@/components/foundation/command-palette';

export function Example() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button variant="outline" onClick={() => setIsOpen(true)}>
        Open palette
      </Button>

      <CommandPalette isOpen={isOpen} onOpenChange={setIsOpen}>
        <CommandPalette.Input placeholder="Search commands…" />
        <CommandPalette.List>
          <CommandPalette.Empty>No results found.</CommandPalette.Empty>
          <CommandPalette.Group heading="Actions">
            <CommandPalette.Item
              shortcut={<KbdGroup keys={['⌘', 'K']} size="sm" />}
              onSelect={() => setIsOpen(false)}
            >
              Search
            </CommandPalette.Item>
          </CommandPalette.Group>
        </CommandPalette.List>
      </CommandPalette>
    </>
  );
}
```

## Accessibility

- Built on `cmdk` for combobox-style filtering, arrow-key navigation, and typeahead.
- Rendered inside a Radix dialog overlay for focus trap, Escape dismiss, and portal rendering.
- **↑ / ↓** — move selection; **Enter** — activate; **Escape** — close and restore focus.
- Includes an sr-only `Dialog.Title` for screen readers.
- Overlay and panel entrance/exit respect `prefers-reduced-motion`.
