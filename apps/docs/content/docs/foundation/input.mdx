## Installation

Copy the component from `packages/ui/src/components/foundation/input/` into your project.

```bash
bun add clsx tailwind-merge
```

## Usage

```tsx
import { Input } from '@/components/foundation/input';

export function Example() {
  return (
    <Input
      label="Email"
      type="email"
      placeholder="you@example.com"
      helperText="We'll never share your email."
      isRequired
    />
  );
}
```

## Variants

- **Default** — a bordered surface at rest; focus swaps the border to the accent color and adds a soft, low-opacity glow (never a hard double ring).
- **Filled** — a quiet `bg-muted` fill with no visible border until focus, when it lifts to `bg-background` with the same accent border + soft glow.
- **Frame** — a gradient-rimmed border (pairs with `Button variant="frame"`) that glows in the brand color on focus.

## States

- **Default** — neutral border, soft accent glow on focus
- **Error** — danger border and matching soft glow, error message below (always visible, not just on focus)
- **Disabled** — reduced opacity, no interaction
- **Addons** — `leftAddon` / `rightAddon` for icons or prefix text
