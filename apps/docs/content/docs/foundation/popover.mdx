## Installation

Copy the component from `packages/ui/src/components/foundation/popover/` into your project.

```bash
bun add @radix-ui/react-popover motion clsx tailwind-merge
```

## Usage

```tsx
import { Popover, Button } from '@/components/foundation/popover';

export function Example() {
  return (
    <Popover>
      <Popover.Trigger asChild>
        <Button variant="outline">Open popover</Button>
      </Popover.Trigger>

      <Popover.Content>
        <div className="space-y-2">
          <h4 className="text-sm font-medium leading-none">Dimensions</h4>
          <p className="text-sm text-muted-foreground">
            Set the width and height for the layer.
          </p>
        </div>
      </Popover.Content>
    </Popover>
  );
}
```

For controlled usage, pass `isOpen` and `onOpenChange` to `Popover` instead of relying on the trigger alone — useful when a save action or async flow should close the panel programmatically.

```tsx
const [isOpen, setIsOpen] = useState(false);

<Popover isOpen={isOpen} onOpenChange={setIsOpen}>
  <Popover.Trigger asChild>
    <Button variant="outline">Edit</Button>
  </Popover.Trigger>
  <Popover.Content>{/* ... */}</Popover.Content>
</Popover>;
```

## Accessibility

Built on `@radix-ui/react-popover` for focus management, portal rendering, and dismiss handling.

| Key | Action |
| --- | --- |
| `Enter` / `Space` | Opens the popover when the trigger is focused. |
| `Esc` | Closes the popover and returns focus to the trigger. |
| `Tab` / `Shift + Tab` | Moves focus inside the panel while open. |

- Focus moves into the panel on open and returns to the trigger on close.
- Clicking outside the panel dismisses it.
- All entrance/exit motion respects `prefers-reduced-motion` — reduced-motion users get an instant state change instead of the scale/fade transition.
