## Installation

Copy the component from `packages/ui/src/components/foundation/select/` into your project.

```bash
bun add @radix-ui/react-select motion clsx tailwind-merge
```

## Usage

```tsx
import { Select } from '@/components/foundation/select';

export function Example() {
  return (
    <Select defaultValue="next">
      <Select.Trigger aria-label="Framework">
        <Select.Value placeholder="Select a framework" />
      </Select.Trigger>
      <Select.Content>
        <Select.Group>
          <Select.Label>Frameworks</Select.Label>
          <Select.Item value="next">Next.js</Select.Item>
          <Select.Item value="react">React</Select.Item>
        </Select.Group>
      </Select.Content>
    </Select>
  );
}
```

## Accessibility

Built on `@radix-ui/react-select`, which implements the [ARIA listbox pattern](https://www.w3.org/WAI/ARIA/apg/patterns/listbox/) for you:

- **Space / Enter** — opens the dropdown, or selects the highlighted item while open
- **Arrow Up / Down** — moves the highlighted item; wraps at the ends
- **Home / End** — jumps to the first / last item
- **Typeahead** — typing letters jumps to the next matching item's label
- **Escape** — closes the dropdown and returns focus to the trigger
- Trigger exposes `role="combobox"` with `aria-expanded` / `aria-controls`; the content is a `role="listbox"` with `aria-selected` on items
- Respects `prefers-reduced-motion` — the open/close, chevron, item stagger, and check-mark animations all drop to an instant transition
