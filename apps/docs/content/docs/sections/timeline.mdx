## Installation

Copy the component from `packages/ui/src/components/sections/timeline/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { Timeline, type TimelineItem } from '@/components/sections/timeline';
import { Flag, Sparkles, Rocket } from 'lucide-react';

const items: TimelineItem[] = [
  {
    date: 'Jan 2024',
    title: 'Project kickoff',
    description: 'Defined the component architecture.',
    badge: 'Milestone',
    icon: <Flag size={14} strokeWidth={1.75} />,
    status: 'done',
  },
  {
    date: 'Jun 2024',
    title: 'Animated layer launched',
    description: 'Added motion components with a11y fallbacks.',
    badge: 'Launch',
    icon: <Sparkles size={14} strokeWidth={1.75} />,
    status: 'current',
  },
  {
    date: 'Dec 2024',
    title: 'Public 1.0',
    description: 'Publish the library under MIT.',
    icon: <Rocket size={14} strokeWidth={1.75} />,
    status: 'upcoming',
  },
];

export function AboutPage() {
  return (
    <Timeline
      eyebrow="Timeline"
      title={
        <>
          How we got{' '}
          <span className="text-brand">here</span>
        </>
      }
      description="Key milestones on the path from idea to shipped product."
      items={items}
    />
  );
}
```

## Accessibility

Milestones are an ordered list (`<ol>`) with a left-rail border and brand dots
marked `aria-hidden` (the text content carries the meaning). Optional icons
render inside the same `aria-hidden` marker — the milestone title and date
still carry the meaning for assistive tech. Dates use semantic `<time>`
elements. Optional badges are plain text spans inside the Badge component.
Scroll-reveal fade+rise per item respects `prefers-reduced-motion`.

| Element | Purpose |
| --- | --- |
| `<section aria-labelledby>` | Landmark with a labelled heading. |
| `<ol>` | Preserves chronological reading order for assistive tech. |
| `<time>` | Semantic date for each milestone. |
| Brand dot (`aria-hidden`) | Decorative — title and date convey the milestone. |
