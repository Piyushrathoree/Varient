## Installation

Copy the component from `packages/ui/src/components/sections/hero-showcase/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { HeroShowcase } from '@/components/sections/hero-showcase';

export function LandingPage() {
  return (
    <HeroShowcase
      title="Showcase your product with depth"
      subtitle="Split hero with a tilted product frame on the right."
      primaryCta={{ label: 'Get started', href: '#start' }}
      secondaryCta={{ label: 'Live preview', href: '#preview' }}
      media={<img src="/dashboard.png" alt="Product screenshot" className="w-full" />}
    />
  );
}
```

## Accessibility

The root is a `<section>` labelled by the h1. The product frame composes `TiltCard` — tilt and the entrance spring are disabled under `prefers-reduced-motion`, falling back to a static perspective frame with hover shadow only. On enter, the frame settles from a slight vertical offset with a single quiet spring (no perpetual float) so it doesn't keep moving while people read the copy.
