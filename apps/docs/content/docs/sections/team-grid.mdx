## Installation

Copy the component from `packages/ui/src/components/sections/team-grid/` into your project.

```bash
bun add motion clsx tailwind-merge @radix-ui/react-avatar
```

## Usage

```tsx
import { TeamGrid } from '@/components/sections/team-grid';

export function AboutPage() {
  return (
    <TeamGrid
      eyebrow="Team"
      title="The people behind Varient"
      description="A small team building copy-paste components."
      members={[
        {
          name: 'Maya Chen',
          role: 'Engineering lead',
          avatarSrc: 'https://i.pravatar.cc/150?img=1',
          socials: { github: 'https://github.com', linkedin: 'https://linkedin.com' },
        },
      ]}
    />
  );
}
```

## Accessibility

The section uses a semantic `<section>` with `aria-labelledby` pointing at the heading.
Each member renders as an `<article>` with heading hierarchy (`h2` section title, `h3` per member).
Social links open in a new tab with `rel="noopener noreferrer"` and descriptive `aria-label` values.

| Key | Action |
| --- | --- |
| `Tab` / `Shift + Tab` | Moves focus between social profile links. |
| `Enter` | Activates the focused social link. |

Scroll-reveal animations are gated behind `prefers-reduced-motion` — reduced-motion users see content immediately without stagger or fade.
