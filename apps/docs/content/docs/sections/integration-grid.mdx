## Installation

Copy the component from `packages/ui/src/components/sections/integration-grid/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { IntegrationGrid } from '@/components/sections/integration-grid';

export function IntegrationsPage() {
  return (
    <IntegrationGrid
      eyebrow="Integrations"
      title="Connect the tools you already use"
      description="Plug into your stack in minutes."
      integrations={[
        {
          name: 'GitHub',
          description: 'Sync issues and pull requests',
          href: 'https://github.com',
        },
        {
          name: 'Linear',
          description: 'Track issues and roadmaps',
          badge: 'New',
          href: 'https://linear.app',
        },
      ]}
    />
  );
}
```

## Accessibility

The section uses a semantic `<section>` with `aria-labelledby` pointing at the heading.
Tiles with `href` render as focusable links with visible focus rings and descriptive `aria-label` values combining name and description.
Each tile carries a visually-hidden `<h3>` with the integration name so screen-reader users can navigate the grid by heading; the matching visible label is `aria-hidden` to avoid double announcement.
Decorative placeholder SVG glyphs are `aria-hidden`.
Outbound links use `rel="noopener noreferrer"`.

Scroll-reveal animations are gated behind `prefers-reduced-motion` — reduced-motion users see content immediately without stagger or fade.
