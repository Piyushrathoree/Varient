## Installation

Copy the component from `packages/ui/src/components/sections/faq/` into your project. It composes the Foundation `Accordion` — copy that too if you do not already have it.

```bash
bun add @radix-ui/react-accordion motion clsx tailwind-merge
```

## Usage

```tsx
import { Faq } from '@/components/sections/faq';

export function Example() {
  return (
    <Faq
      eyebrow="FAQ"
      title="Frequently asked questions"
      description="Quick answers about copying components and theming."
      items={[
        {
          question: 'Is Varient free?',
          answer: 'Yes — MIT licensed with no paid tier.',
        },
        {
          question: 'Do I install a package?',
          answer: 'No. Copy the source from the docs and it lives in your project.',
        },
      ]}
    />
  );
}
```

## Accessibility

Composes the Foundation `Accordion`, which is built on `@radix-ui/react-accordion` for correct roles and keyboard behavior.

| Key | Action |
| --- | --- |
| `Enter` / `Space` | Toggles the focused item open or closed. |
| `Tab` / `Shift + Tab` | Moves focus to the next / previous focusable element. |
| `Arrow Down` | Moves focus to the next trigger. |
| `Arrow Up` | Moves focus to the previous trigger. |
| `Home` | Moves focus to the first trigger. |
| `End` | Moves focus to the last trigger. |

The section header uses scroll-triggered motion that collapses to an instant reveal when `prefers-reduced-motion` is enabled.
