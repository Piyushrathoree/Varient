## Installation

Copy the component from `packages/ui/src/components/sections/not-found-page/` into your project.

```bash
bun add motion clsx tailwind-merge @radix-ui/react-slot
```

## Usage

```tsx
import { NotFoundPage } from '@/components/sections/not-found-page';

export function NotFound() {
  return (
    <NotFoundPage
      code="404"
      title="Page not found"
      description="The page you're looking for doesn't exist or has been moved."
      homeHref="/"
      homeLabel="Back to home"
      secondaryAction={{ label: 'Browse components', href: '/docs' }}
      popularLinks={[
        { label: 'Docs', href: '/docs' },
        { label: 'Status', href: '/status' },
        { label: 'Support', href: '/support' },
      ]}
    />
  );
}
```

## Accessibility

The root is a `<section>` with `aria-labelledby` pointing at the `<h1>` title.
The status code is visible text — screen readers announce it before the title.
A quiet, static `DotPattern` sits behind the code as `aria-hidden` decoration and never carries content.
Primary and secondary actions render as real `<a>` elements via `Button asChild`, preserving native link semantics and keyboard activation.
`popularLinks`, when provided, render inside a labeled `<nav aria-label="Popular links">` with real `<a>` elements and visible `focus-visible` rings.
Focus rings follow the shared Button focus-visible styles.

Entrance animation is a single fade-and-rise on the content column, gated behind `prefers-reduced-motion` — reduced-motion users see the layout immediately with no motion. The background dot pattern is static by default (no perpetual loop), so there is nothing to pause offscreen.

| Key | Action |
| --- | --- |
| `Tab` / `Shift + Tab` | Moves focus between the primary, secondary, and popular-link actions. |
| `Enter` | Activates the focused link. |
