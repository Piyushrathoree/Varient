## Installation

Copy the component from `packages/ui/src/components/sections/footer/` into your project.

```bash
bun add clsx tailwind-merge
```

## Usage

```tsx
import { Footer } from '@/components/sections/footer';

export function Example() {
  return (
    <Footer
      tagline="Copy-paste React components. You own the code."
      statusText="All systems operational"
      columns={[
        {
          title: 'Library',
          links: [
            { label: 'Components', href: '/components' },
            { label: 'Foundation', href: '/components?layer=foundation' },
          ],
        },
        {
          title: 'Guide',
          links: [{ label: 'Introduction', href: '/docs/getting-started/introduction' }],
        },
        {
          title: 'Project',
          links: [{ label: 'GitHub', href: 'https://github.com/example', isExternal: true }],
        },
      ]}
      socialLinks={[
        { label: 'GitHub', href: 'https://github.com/example', platform: 'github' },
        { label: 'X', href: 'https://x.com/example', platform: 'x' },
      ]}
    />
  );
}
```

## Accessibility

| Element | Behavior |
| --- | --- |
| Wordmark link | `aria-label` identifies the home link; visible focus ring on keyboard focus. |
| Link columns | Each column is a `<nav>` with an `aria-label` matching the column title. |
| Social icons | Each icon link has a descriptive `aria-label`; decorative SVGs are `aria-hidden`. |
| Status pill | Decorative ping animation is `aria-hidden`; the label text is readable by assistive tech. |
| Credit line | External SmoothUI link opens in a new tab with `rel="noopener noreferrer"`. |

External links use `target="_blank"` with `rel="noopener noreferrer"`. The status ping respects `prefers-reduced-motion` via `motion-reduce:animate-none`.
