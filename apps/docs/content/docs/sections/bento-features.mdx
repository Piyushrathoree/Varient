## Installation

Copy the component from `packages/ui/src/components/sections/bento-features/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { BentoFeatures } from '@/components/sections/bento-features';

export function LandingPage() {
  return (
    <BentoFeatures
      eyebrow="Features"
      title={
        <>
          Built for teams who{' '}
          <span className="text-brand">ship fast</span>
        </>
      }
      description="An asymmetric bento grid that highlights what makes your product different."
      items={[
        {
          title: 'Token-driven theming',
          description: 'One CSS file restyles every component.',
          className: 'md:col-span-2',
        },
        {
          title: 'Copy-paste DX',
          description: 'Copy the source and own the code.',
          className: 'md:row-span-2',
        },
        {
          title: 'Motion built in',
          description: 'Springs with reduced-motion fallbacks.',
        },
      ]}
    />
  );
}
```

## Accessibility

Each feature card uses the Foundation `Card` surface inside a semantic `<section>`
with a labelled heading. Visual placeholders are `aria-hidden`. Cards lift on
hover via `isHoverable`. Scroll-reveal stagger on cards respects
`prefers-reduced-motion`.

| Key | Action |
| --- | --- |
| `Tab` / `Shift + Tab` | Moves focus through any interactive elements inside custom visual slots. |

No keyboard traps — the default placeholders are decorative only.
