## Installation

Copy the component from `packages/ui/src/components/sections/logo-cloud/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { LogoCloud } from '@/components/sections/logo-cloud';

export function LandingPage() {
  return (
    <LogoCloud
      title="Trusted by teams at"
      logos={[
        { name: 'Acme' },
        { name: 'Northwind' },
        { name: 'Vertex' },
      ]}
    />
  );
}
```

## Marquee

Pass `isMarquee` to wrap the row in a self-contained CSS translate loop. Under `prefers-reduced-motion`, the row stays static regardless of this flag.

```tsx
<LogoCloud isMarquee title="Powering products worldwide" />
```

## Accessibility

The root is a `<section>` with `aria-label` from the title (or a generic partner label when title is omitted). Decorative SVG marks are `aria-hidden`. Logo names are visible text — no image-only logos without accompanying text.

Marquee animation is disabled under `prefers-reduced-motion`. The duplicated marquee track is marked `aria-hidden` so screen readers do not hear logos twice.
