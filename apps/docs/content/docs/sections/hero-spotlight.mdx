## Installation

Copy the component from `packages/ui/src/components/sections/hero-spotlight/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { HeroSpotlight } from '@/components/sections/hero-spotlight';

export function LandingPage() {
  return (
    <HeroSpotlight
      title="Ship interfaces that feel alive"
      highlightWord="alive"
      subtitle="Dark hero with sweeping spotlight beam and dual CTAs."
      announcement={{ badge: 'New', label: 'Hero Spotlight is live', href: '/components' }}
      primaryCta={{ label: 'Browse components', href: '/components' }}
      secondaryCta={{ label: 'Read docs', href: '/docs' }}
    />
  );
}
```

## Accessibility

The root is a `<section>` labelled by the h1 via `aria-labelledby`. CTA links are native anchors inside `Button asChild`. The sweeping spotlight beam is decorative (`aria-hidden`) and does not intercept pointer events.

Under `prefers-reduced-motion`, the beam is static, entrance motion is instant, and the announcement shimmer is omitted.
