## Installation

Copy the component from `packages/ui/src/components/sections/hero-grid/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { HeroGrid } from '@/components/sections/hero-grid';

export function LandingPage() {
  return (
    <HeroGrid
      title="Precision grids, premium motion"
      subtitle="Fine line-grid canvas with radial fade and staggered entrance."
      primaryCta={{ label: 'Start building', href: '#start' }}
      secondaryCta={{ label: 'See examples', href: '#examples' }}
      stats={[
        { value: '75+', label: 'Components' },
        { value: '3', label: 'Layers' },
        { value: '0', label: 'Runtime deps' },
      ]}
    />
  );
}
```

## Accessibility

The root is a `<section>` labelled by the h1. Background grid, fade mask, and glow orbs (or a custom `backgroundSlot`) are decorative (`aria-hidden`). Text entrance respects `prefers-reduced-motion` — content renders immediately without stagger when reduced motion is preferred, and glow orbs render as static, motionless fills.
