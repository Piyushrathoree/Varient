## Installation

Copy the component from `packages/ui/src/components/sections/comparison-table/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { ComparisonTable } from '@/components/sections/comparison-table';

export function PricingPage() {
  return (
    <ComparisonTable
      eyebrow="Compare plans"
      title={
        <>
          Pick the plan that{' '}
          <span className="text-brand">fits your team</span>
        </>
      }
      description="See exactly what each tier includes before you commit."
      plans={[
        { name: 'Starter', price: 'Free', cta: { label: 'Get started' } },
        {
          name: 'Pro',
          price: '$29/mo',
          isHighlighted: true,
          cta: { label: 'Start trial' },
        },
        { name: 'Enterprise', price: 'Custom', cta: { label: 'Contact sales' } },
      ]}
      rows={[
        { category: 'Core', feature: 'Foundation components', values: [true, true, true] },
        { feature: 'Animated layer', values: [false, true, true] },
        { feature: 'Team members', values: ['1', '10', 'Unlimited'] },
      ]}
      footnote="Prices shown in USD, billed monthly."
    />
  );
}
```

## Accessibility

The table uses proper `scope` attributes: `scope="col"` on plan headers and
`scope="row"` on feature names. Boolean values expose screen-reader text
("Included" / "Not included"). The table wrapper scrolls horizontally on
narrow viewports via `overflow-x-auto`.

| Element | Attribute | Purpose |
| --- | --- | --- |
| Feature column header | `scope="col"` | Identifies the features column for assistive tech. |
| Plan column headers | `scope="col"` | Associates each data cell with its plan. |
| Feature name cells | `scope="row"` | Associates each value cell with its feature. |
| Check/cross icons | `sr-only` label | Announces inclusion state for boolean values. |

CTA buttons are real `<button>` elements (or `<a>` via `asChild`) with visible
focus rings. Scroll-reveal on the section header respects `prefers-reduced-motion`.
