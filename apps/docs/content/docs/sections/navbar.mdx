## Installation

Copy the component from `packages/ui/src/components/sections/navbar/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { Navbar } from '@/components/sections/navbar';

export function SiteShell() {
  return (
    <Navbar
      brandName="Varient"
      isSticky={false}
      items={[
        { label: 'Components', href: '/components' },
        { label: 'Docs', href: '/docs' },
        { label: 'Sections', href: '/docs/sections/hero' },
      ]}
      githubHref="https://github.com/piyush/varient"
      cta={{ label: 'Get started', href: '/docs/getting-started/installation' }}
    />
  );
}
```

## Anatomy

Per the app shell spec: a sticky top wrapper with an inner floating pill (`rounded-full border border-border bg-background/80 backdrop-blur-xl`). Left to right — two-tone wordmark (or custom `logo`), nav links, then optional GitHub icon and CTA button. Below `md`, links collapse behind a hamburger that toggles a slide-down panel animated with height and opacity.

Pass `logo` to replace the default two-tone wordmark. Set `isSticky={false}` when embedding inside a docs preview frame.

## Accessibility

The root is a `<header>`. Desktop links sit in a `<nav aria-label="Main">`. The mobile menu button exposes `aria-expanded` and `aria-controls`; the panel is labelled `aria-label="Mobile"`. Press `Escape` to close the mobile menu.

| Key | Action |
| --- | --- |
| `Tab` / `Shift + Tab` | Moves focus through the wordmark, nav links, GitHub icon, CTA, and menu button. |
| `Enter` / `Space` | Activates the focused link or button. |
| `Escape` | Closes the mobile menu when open. |

Slide-down panel animation respects `prefers-reduced-motion` — reduced-motion users get an instant open/close without height or opacity transitions.
