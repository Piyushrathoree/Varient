## Installation

Copy the component from `packages/ui/src/components/sections/hero/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { Hero } from '@/components/sections/hero';

export function LandingPage() {
  return (
    <Hero
      title="Animated components for web and native"
      subtitle="Copy-paste React components with real motion built in."
      announcement={{
        badge: '25+',
        label: 'components shipped, more every week',
        href: '/components',
      }}
      primaryCta={{ label: 'Explore components', href: '/components' }}
      secondaryCta={{ label: 'Read the docs', href: '/docs' }}
      builtWith={['Next.js', 'React', 'Tailwind CSS', 'Motion']}
    />
  );
}
```

## Anatomy

Per the landing spec: two columns on `lg` (`lg:grid-cols-2`). The left column carries an announcement pill, display headline, subhead, primary + outline CTAs, and an optional "built with" row. The right column renders a grid of real foundation components (Card, Badge, Switch, Input, NumberTicker) — the product demos itself, not a screenshot.

Pass `visual` to replace the default showcase with your own node. Pass `builtWith={[]}` to hide the tech stack row, and `announcement={null}` to hide the pill entirely (omitting the prop still renders the default announcement).

The announcement pill's shimmer sweeps once on mount, then replays once per hover — it never loops perpetually. The visual showcase grid collapses to a single column below `sm` so it stays legible on narrow phones.

## Accessibility

The root is a `<section>` labelled by the h1 via `aria-labelledby`. CTA links are native anchors inside `Button asChild` so they remain keyboard-focusable with visible focus rings.

| Key | Action |
| --- | --- |
| `Tab` / `Shift + Tab` | Moves focus through the announcement link, CTA buttons, and interactive elements in the visual grid. |
| `Enter` / `Space` | Activates the focused link or button. |

Entrance animations respect `prefers-reduced-motion` — reduced-motion users see content immediately without translate or fade delays. The announcement shimmer is omitted when reduced motion is preferred, and otherwise plays once on mount plus once per hover rather than looping forever.
