## Installation

Copy the component from `packages/ui/src/components/animated/spotlight/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Place `Spotlight` inside a `relative` container. Content sits above the glow with
`relative z-10`:

```tsx
import { Spotlight } from '@/components/animated/spotlight';

export function Example() {
  return (
    <div className="relative overflow-hidden rounded-xl border border-border bg-foreground p-8">
      <Spotlight />
      <div className="relative z-10">
        <h3 className="text-lg font-semibold text-background">Hover the card</h3>
      </div>
    </div>
  );
}
```

## Border variant

Pass `variant="border"` to trade the free-floating glow for one that hugs the
container's edge — the pointer position is projected onto the nearest point
on the perimeter, so the light travels along the border instead of floating
across the surface:

```tsx
<Spotlight variant="border" borderWidth={3} />
```

## Accessibility

- The overlay is decorative — it is marked `aria-hidden="true"` and uses
  `pointer-events-none`, so it never intercepts clicks or focus.
- Parent content remains fully interactive; stack content above the glow with
  `relative z-10`.
- Under `prefers-reduced-motion`, pointer tracking and hover fade are disabled.
  A static, centered soft glow is shown instead with no cursor follow.
- On coarse/non-hover pointers (touch), tracking listeners never attach and the
  same static soft-glow treatment is shown instead.
- Tracking listeners attach to the parent element only when motion is allowed
  and the primary pointer is fine (mouse/trackpad), and are removed on unmount.
