## Installation

Copy the component from `packages/ui/src/components/animated/animated-list/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Compose notification-style rows with `AnimatedListItem`. The list cycles children on an interval — newest on top:

```tsx
import { AnimatedList, AnimatedListItem } from '@/components/animated/animated-list';

export function Example() {
  return (
    <AnimatedList interval={2500}>
      <AnimatedListItem>
        <p className="text-sm font-medium">Deployment ready</p>
      </AnimatedListItem>
      <AnimatedListItem>
        <p className="text-sm font-medium">New comment on PR</p>
      </AnimatedListItem>
    </AnimatedList>
  );
}
```

## Accessibility

- The list container uses `aria-live="polite"` so screen readers are notified of new entries without interrupting.
- A screen-reader-only summary tracks which item in the cycle is currently shown.
- Under `prefers-reduced-motion`, all items render statically in document order with no cycling animation.

## Notes

- The cycling interval automatically pauses while the list is scrolled offscreen (via `IntersectionObserver`) and resumes when it re-enters the viewport, so background tabs and distant sections don't keep timers running.
- Item collection identifies `AnimatedListItem` children via an internal marker rather than `displayName`, so wrapping the item in a memoized or higher-order component won't silently break the cycle.
