## Installation

Copy the component from `packages/ui/src/components/animated/gradient-text/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { GradientText } from '@/components/animated/gradient-text';

export function Example() {
  return (
    <GradientText className="font-title text-4xl font-semibold tracking-tight">
      Ship faster
    </GradientText>
  );
}
```

## Accessibility

Gradient text is decorative headline styling — the string in `children` is plain text in the DOM and is read normally by assistive tech. When `prefers-reduced-motion` is set, the gradient renders as a static fill with no background-position animation; the injected keyframes are omitted entirely. The sweep also pauses automatically while the element is scrolled offscreen, so it only burns CPU when visible. Prefer modest durations and high-contrast stop colors so the text remains readable in both light and dark themes.
