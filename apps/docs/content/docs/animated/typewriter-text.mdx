## Installation

Copy the component from `packages/ui/src/components/animated/typewriter-text/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { TypewriterText } from '@/components/animated/typewriter-text';

export function Example() {
  return (
    <h1 className="text-3xl font-semibold text-foreground">
      Built for{' '}
      <TypewriterText
        text={['designers', 'developers', 'teams']}
        className="text-brand"
      />
    </h1>
  );
}
```

## Accessibility

The animating span (typed characters and cursor) is `aria-hidden` — a rapidly updating string is not something a screen reader should narrate character by character. The wrapper exposes the full intended message through `aria-label` (all phrases joined when `text` is an array). When `prefers-reduced-motion` is set, the first phrase renders immediately with no typing, deleting, or cursor blink animation. Timeouts are cleared on unmount so cycling never leaks timers.
