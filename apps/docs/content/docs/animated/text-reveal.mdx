## Installation

Copy the component from `packages/ui/src/components/animated/text-reveal/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { TextReveal } from '@/components/animated/text-reveal';

export function Example() {
  return (
    <TextReveal
      text="Words reveal as you scroll into view."
      className="text-2xl font-semibold text-foreground"
    />
  );
}
```

## Accessibility

Individual animating words are `aria-hidden` — a partially-revealed string is not something screen readers should narrate word-by-word. The complete string is always present through a visually-hidden (`sr-only`) span, so assistive tech reads the full text once, correctly, regardless of animation state. When `prefers-reduced-motion` is set, every word renders immediately at full opacity with no translate. The animation fires once, the first time the block scrolls into view (`useInView` with `once: true` and a `-80px` margin) — it does not replay on later scrolls.
