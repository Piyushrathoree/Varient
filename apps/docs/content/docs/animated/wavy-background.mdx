## Installation

Copy the component from `packages/ui/src/components/animated/wavy-background/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

`WavyBackground` fills its container and draws waves on a canvas layer behind
children:

```tsx
import { WavyBackground } from '@/components/animated/wavy-background';

export function Example() {
  return (
    <WavyBackground className="min-h-64 rounded-xl border border-border" speed="slow">
      <div className="flex flex-col items-center justify-center gap-3 py-16 text-center">
        <h2 className="text-2xl font-semibold">Wavy backdrop</h2>
      </div>
    </WavyBackground>
  );
}
```

## Accessibility

- The canvas is decorative — it is marked `aria-hidden="true"` and uses
  `pointer-events-none`, so it never intercepts clicks or focus.
- Children render in a `relative z-10` wrapper and remain fully interactive.
- Under `prefers-reduced-motion`, a single static frame is drawn and the
  requestAnimationFrame loop is not started.
- The animation loop pauses automatically while the component is scrolled
  offscreen (via `IntersectionObserver`) and resumes when it re-enters the
  viewport, so background instances don't burn CPU while not visible.
- A `ResizeObserver` keeps the canvas sized to the container and cleans up on
  unmount.
