## Installation

Copy the component from `packages/ui/src/components/animated/sliding-number/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { SlidingNumber } from '@/components/animated/sliding-number';

export function Example() {
  return <SlidingNumber value={128} />;
}
```

Compose separators and prefixes/suffixes around it as plain text — the component only owns the
rolling digits:

```tsx
<div className="flex items-baseline gap-1">
  <span>$</span>
  <SlidingNumber value={1908} isGrouped />
  <span className="text-muted-foreground">/yr</span>
</div>
```

## Accessibility

The rolling digit columns live inside an `aria-hidden` wrapper — a mid-roll digit is not something
a screen reader should narrate. The fully formatted value (including sign, grouping, and decimals)
is always available through a visually-hidden `aria-live="polite"` span, so assistive tech announces
the settled value once it changes rather than every intermediate frame. When
`prefers-reduced-motion` is set, digits swap instantly with no roll or pop animation — including the
column that pops in or out when the digit count grows or shrinks (e.g. 99 → 100). `tabular-nums` is
applied throughout so digit columns never jitter in width.
