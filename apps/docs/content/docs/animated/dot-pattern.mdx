## Installation

Copy the component from `packages/ui/src/components/animated/dot-pattern/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

`DotPattern` mirrors the `.frame-box` dotted canvas. Pass a radial mask via
`className` to fade dots at the edges:

```tsx
import { DotPattern } from '@/components/animated/dot-pattern';

export function Example() {
  return (
    <div className="relative h-56 overflow-hidden rounded-xl border border-border bg-primary">
      <DotPattern
        isGlowing
        className="[mask-image:radial-gradient(ellipse_at_center,black_20%,transparent_75%)]"
      />
    </div>
  );
}
```

## Accessibility

- Decorative SVG layer is `aria-hidden` with `pointer-events-none`.
- Under `prefers-reduced-motion`, the glow wave is disabled and dots render
  as a static pattern.
