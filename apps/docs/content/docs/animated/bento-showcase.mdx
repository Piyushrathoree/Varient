## Installation

Copy the component from `packages/ui/src/components/animated/bento-showcase/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Compose cells with `colSpan` / `rowSpan`, mount animated backgrounds per card, and reveal a footer CTA on hover:

```tsx
import { BentoShowcase, BentoShowcaseCard } from '@/components/animated/bento-showcase';

export function Example() {
  return (
    <BentoShowcase>
      <BentoShowcaseCard
        colSpan={2}
        title="Wide feature"
        description="Background slot + hover footer."
        background={<div className="h-full bg-gradient-to-br from-brand/15 to-muted" />}
        footer={<span className="text-sm font-medium text-brand">Learn more →</span>}
      />
      <BentoShowcaseCard title="Standard cell" description="Single column." />
    </BentoShowcase>
  );
}
```

## Accessibility

- Cards use semantic heading (`h3`) and paragraph text for screen readers.
- Hover-reveal footers are always visible under `prefers-reduced-motion` via `motion-reduce` utilities.
- Group dimming and staggered entrance are disabled under reduced motion — cells render statically at full opacity.
