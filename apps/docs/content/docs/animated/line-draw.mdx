## Installation

Copy the component from `packages/ui/src/components/animated/line-draw/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Pick a built-in preset or pass custom path data. Drawing triggers when the SVG scrolls into view:

```tsx
import { LineDraw } from '@/components/animated/line-draw';

export function Example() {
  return (
    <div className="space-y-6">
      <LineDraw preset="underline" className="h-6 w-full" duration={1.2} />
      <LineDraw preset="arrow" className="h-12 w-48" isLooping />
      <LineDraw
        path="M 0 50 L 100 50"
        viewBox="0 0 100 100"
        className="size-24"
      />
    </div>
  );
}
```

## Accessibility

- Decorative SVGs are marked `aria-hidden` — pair with visible text when the path conveys meaning.
- Under `prefers-reduced-motion`, paths render fully drawn with no stroke animation.
- Presets use `strokeLinecap="round"` for smoother decorative strokes at small sizes.
