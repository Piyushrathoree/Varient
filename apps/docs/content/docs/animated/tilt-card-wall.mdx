## Installation

Copy the component from `packages/ui/src/components/animated/tilt-card-wall/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Pass an array of testimonial items. Each card fans from the center at rest and
springs upright when hovered or focused:

```tsx
import { TiltCardWall } from '@/components/animated/tilt-card-wall';

const items = [
  {
    quote: 'A small tool with a big payoff for our design workflow.',
    name: 'Sarah Chen',
    role: 'Product Hunt review',
    rating: 5.0,
    source: 'Product Hunt',
  },
  {
    quote: 'Crisp overlay, fast, and never gets in the way.',
    name: 'Marcus Webb',
    role: 'Frontend Developer',
    rating: 5.0,
    source: 'Chrome Web Store',
  },
];

export function Example() {
  return <TiltCardWall items={items} maxRotation={12} />;
}
```

## Accessibility

- Each card is a focusable `<article>` with a visible focus ring. Keyboard
  focus triggers the same straighten-and-lift interaction as hover.
- Neighboring cards dim slightly when one card is active so the focused card
  reads clearly.
- Under `prefers-reduced-motion`, the fan layout stays static and hover/focus
  only adjusts opacity and shadow — no rotation, scale, or lift animation.
- On viewports below `640px`, cards stack vertically with no fan rotation so
  the wall stays readable at 375px.
