## Installation

Copy the component from `packages/ui/src/components/animated/orbit/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Place `Orbit` around a central anchor in a relative container. Pass orbiting
items as children — they are distributed evenly around the circle:

```tsx
import { Orbit } from '@/components/animated/orbit';

export function Example() {
  return (
    <div className="relative flex h-64 items-center justify-center">
      <div className="absolute z-10 size-12 rounded-full border border-border bg-card" />
      <Orbit radius={80} duration={20}>
        <div className="size-8 rounded-lg border border-border bg-background" />
        <div className="size-8 rounded-lg border border-border bg-background" />
        <div className="size-8 rounded-lg border border-border bg-background" />
      </Orbit>
    </div>
  );
}
```

## Accessibility

- The optional path ring is decorative and marked `aria-hidden` with `pointer-events-none`.
- Under `prefers-reduced-motion`, items are statically distributed around the circle with no rotation animation.
- Orbiting children remain in the tab order and can receive focus if they are interactive elements.
