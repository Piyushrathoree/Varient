## Installation

Copy the component from `packages/ui/src/components/animated/meteors/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

`Meteors` is a decorative layer. Wrap your content in a `relative overflow-hidden` container and keep interactive elements above the effect with `relative z-10`:

```tsx
import { Meteors } from '@/components/animated/meteors';

export function Example() {
  return (
    <div className="relative overflow-hidden rounded-xl bg-foreground px-6 py-16">
      <Meteors number={20} />
      <h2 className="relative z-10 text-center text-lg font-semibold text-background">
        Meteor shower
      </h2>
    </div>
  );
}
```

## Accessibility

- The layer is `pointer-events-none` and `aria-hidden` on each meteor — it never intercepts clicks or receives focus.
- Decorative only; do not rely on motion to convey information.
- Under `prefers-reduced-motion` the animation loop is disabled and a few static meteors are rendered instead, preserving the look without perpetual motion.
- The loop pauses automatically while the layer is scrolled offscreen and resumes on re-entry, so mounted-but-unseen instances don't burn CPU.
