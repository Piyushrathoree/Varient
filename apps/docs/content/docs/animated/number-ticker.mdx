## Installation

Copy the component from `packages/ui/src/components/animated/number-ticker/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { NumberTicker } from '@/components/animated/number-ticker';

export function Example() {
  return <NumberTicker value={100} suffix="%" />;
}
```

## Accessibility

The animating digits are `aria-hidden` — a rapidly-updating number is not something a screen reader should narrate mid-count. The final formatted value (prefix + value + suffix) is always present through a visually-hidden (`sr-only`) span, so assistive tech reads it once, correctly, regardless of animation state. When `prefers-reduced-motion` is set, the count-up is skipped entirely and the final value renders immediately. The animation fires once, the first time the component scrolls into view (`useInView` with `once: true` and a `-80px` margin) — it does not replay on later scrolls.
