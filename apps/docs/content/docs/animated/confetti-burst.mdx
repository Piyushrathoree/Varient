## Installation

Copy the component from `packages/ui/src/components/animated/confetti-burst/` into your project.

```bash
bun add canvas-confetti motion clsx tailwind-merge
```

Also install TypeScript types for canvas-confetti:

```bash
bun add -d @types/canvas-confetti
```

## Usage

`ConfettiBurst` renders a styled button that fires confetti from its center on
click. Use `fireConfetti()` or `useConfetti()` for imperative bursts elsewhere:

```tsx
import { ConfettiBurst, fireConfetti } from '@/components/animated/confetti-burst';

export function Example() {
  return (
    <>
      <ConfettiBurst onFire={() => console.log('fired!')}>
        Celebrate
      </ConfettiBurst>

      <button type="button" onClick={() => fireConfetti({ originX: 0.5, originY: 0.5 })}>
        Fire from center
      </button>
    </>
  );
}
```

## Accessibility

- Renders a real `<button>` element with keyboard focus, `Enter`/`Space`
  activation, and the standard visible focus ring.
- Under `prefers-reduced-motion` no confetti particles are emitted — the
  click handler and `onFire` callback still run, but the visual burst is
  skipped entirely. `useConfetti().fire()` is also a no-op in this mode.
- `canvas-confetti` is configured with `disableForReducedMotion: true` as an
  additional safeguard for imperative calls.
