## Installation

Copy the component from `packages/ui/src/components/animated/grid-pattern/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Place `GridPattern` inside a `relative overflow-hidden` container — it renders
`absolute inset-0` and fills the parent:

```tsx
import { GridPattern } from '@/components/animated/grid-pattern';

export function Example() {
  return (
    <div className="relative h-56 overflow-hidden rounded-xl border border-border bg-background">
      <GridPattern squares={[[2, 1], [4, 3]]} />
      <p className="relative z-10 p-4 font-semibold">Your content</p>
    </div>
  );
}
```

## Accessibility

- Decorative SVG layer is `aria-hidden` with `pointer-events-none` so content
  above stays fully interactive.
- Under `prefers-reduced-motion`, animation is disabled: highlighted cells
  render as static fills and the grid does not pan.
