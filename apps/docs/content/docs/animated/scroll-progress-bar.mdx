## Installation

Copy the component from `packages/ui/src/components/animated/scroll-progress-bar/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Place at the top or bottom of the viewport (or inside a scroll container for demos):

```tsx
import { useRef } from 'react';
import { ScrollProgressBar } from '@/components/animated/scroll-progress-bar';

export function Example() {
  const scrollRef = useRef<HTMLDivElement>(null);

  return (
    <>
      <ScrollProgressBar />
      {/* or, for a contained scroll area: */}
      <div ref={scrollRef} className="relative h-96 overflow-y-auto">
        <ScrollProgressBar isFixed={false} scrollContainerRef={scrollRef} />
        {/* content */}
      </div>
    </>
  );
}
```

## Accessibility

- Exposes `role="progressbar"` with `aria-valuemin`, `aria-valuemax`, and a live `aria-valuenow` synced to scroll percentage.
- Under `prefers-reduced-motion` the fill binds directly to scroll position (no spring smoothing) — the bar remains functional because it conveys reading progress, not decoration.
