## Installation

Copy the component from `packages/ui/src/components/animated/word-rotate/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { WordRotate } from '@/components/animated/word-rotate';

export function Example() {
  return (
    <h1 className="text-3xl font-semibold text-foreground">
      Build something <WordRotate words={['beautiful', 'delightful', 'accessible']} />
    </h1>
  );
}
```

## Accessibility

The outer wrapper is `aria-live="polite"` and `aria-atomic="true"`, so each word change is announced without interrupting the user. When `prefers-reduced-motion` is set, the cycling interval itself is paused and the current word is shown with no opacity or translate animation. Only `transform` and `opacity` animate in the default motion path, keeping the transition GPU-friendly and predictable.
