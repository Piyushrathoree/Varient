## Installation

Copy the component from `packages/ui/src/components/animated/animated-beam/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

`AnimatedBeam` renders an absolute SVG overlay inside a container. Pass refs to
the container and both endpoint elements — the beam tracks their centers and
stays in sync on resize:

```tsx
'use client';

import { useRef } from 'react';
import { AnimatedBeam } from '@/components/animated/animated-beam';

export function Example() {
  const containerRef = useRef<HTMLDivElement>(null);
  const fromRef = useRef<HTMLDivElement>(null);
  const toRef = useRef<HTMLDivElement>(null);

  return (
    <div ref={containerRef} className="relative flex items-center justify-between p-12">
      <div ref={fromRef} className="size-12 rounded-full border border-border bg-background" />
      <div ref={toRef} className="size-12 rounded-full border border-border bg-card" />
      <AnimatedBeam containerRef={containerRef} fromRef={fromRef} toRef={toRef} />
    </div>
  );
}
```

## Accessibility

- The SVG overlay is decorative and marked `aria-hidden` — it does not affect the accessible name or structure of connected elements.
- Under `prefers-reduced-motion`, the traveling gradient is disabled and only a static base path is shown.
- The beam layer uses `pointer-events-none` so endpoint elements remain fully interactive.
