## Installation

Copy the component from `packages/ui/src/components/animated/particles/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Place `Particles` inside a `relative overflow-hidden` container. The canvas
fills the container as a decorative background layer:

```tsx
import { Particles } from '@/components/animated/particles';

export function Example() {
  return (
    <div className="relative h-64 overflow-hidden rounded-xl border border-border bg-card">
      <Particles />
      <h3 className="relative z-10 flex h-full items-center justify-center text-lg font-semibold">
        Your headline
      </h3>
    </div>
  );
}
```

## Accessibility

- The canvas is decorative and marked `aria-hidden` — content layered above carries the accessible name.
- Under `prefers-reduced-motion`, static scattered dots are drawn once with no animation loop and no pointer tracking.
- The canvas uses `pointer-events-none` so foreground content remains fully interactive; pointer position is read from the parent container.
