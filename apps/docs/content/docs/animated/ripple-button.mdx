## Installation

Copy the component from `packages/ui/src/components/animated/ripple-button/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

`RippleButton` is a drop-in animated button — pass `variant` and `size` the
same way as the Foundation `Button`:

```tsx
import { RippleButton } from '@/components/animated/ripple-button';

export function Example() {
  return (
    <RippleButton variant="primary" rippleColor="color-mix(in oklab, white 40%, transparent)">
      Click me
    </RippleButton>
  );
}
```

## Accessibility

- Renders a real `<button>` element with keyboard focus, `Enter`/`Space`
  activation, and the standard visible focus ring.
- Under `prefers-reduced-motion` the expanding ripple is replaced by a brief
  background flash — the button remains fully functional with no motion
  circles rendered.
- Ripple overlays are `aria-hidden` and do not interfere with screen reader
  announcements of the button label.
