## Installation

Copy the component from `packages/ui/src/components/animated/shimmer-button/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { ShimmerButton } from '@/components/animated/shimmer-button';

export function Example() {
  return <ShimmerButton size="lg">Get started</ShimmerButton>;
}
```

## Accessibility

- Renders a real `<button>` element — keyboard focus, `Enter`/`Space` activation, and the visible focus ring work without extra markup.
- Press feedback uses `active:scale-[0.97]` with a transform-only transition, matching the house button chassis.
- Under `prefers-reduced-motion` the shimmer loop is not mounted; the button remains a fully functional static CTA with the same disabled and focus behavior.
- The shimmer sweep pauses automatically while the button is scrolled offscreen, avoiding wasted animation work.
