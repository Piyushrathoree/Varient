## Installation

Copy the component from `packages/ui/src/components/animated/text-scramble/` into your project.

```bash
bun add clsx tailwind-merge motion
```

## Usage

```tsx
import { TextScramble } from '@/components/animated/text-scramble';

export function Example() {
  return (
    <h1 className="font-mono text-3xl font-semibold text-foreground">
      <TextScramble text="Decrypt the message" />
    </h1>
  );
}
```

## Accessibility

The wrapper element carries `aria-label={text}`, so assistive tech always gets the final, correct string regardless of animation state — the animating overlay span is `aria-hidden`. Layout stability comes from rendering the real text twice: an invisible sizer that reserves the box, and an absolutely-positioned overlay that actually animates, so surrounding layout never jitters as glyphs of different widths cycle through. Pair with a monospace font (e.g. `font-mono`) for the cleanest column alignment while shuffling. Under `prefers-reduced-motion`, the final text renders immediately with no shuffle. The interval timer is always cleared on unmount or when `text` changes.
