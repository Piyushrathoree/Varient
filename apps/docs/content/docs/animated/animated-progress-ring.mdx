## Installation

Copy the component from `packages/ui/src/components/animated/animated-progress-ring/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Pass a `value` from 0–100. The ring animates once when scrolled into view and
re-animates whenever `value` changes:

```tsx
import { AnimatedProgressRing } from '@/components/animated/animated-progress-ring';

export function Example() {
  return <AnimatedProgressRing value={72} size={120} strokeWidth={8} />;
}
```

## Accessibility

- The wrapper exposes `role="progressbar"` with `aria-valuemin={0}`,
  `aria-valuemax={100}`, and `aria-valuenow` set to the current rounded value.
- An `aria-label` describes the current progress percentage for screen readers.
- The SVG arc and the center digits are always `aria-hidden` — the accessible
  value lives solely on the progressbar's `aria-valuenow`/`aria-label`, so it
  is announced whether or not `showValue` renders the visual digits.
- Under `prefers-reduced-motion` the ring and count jump directly to the
  target value with no sweep animation.
