## Installation

Copy the component from `packages/ui/src/components/animated/retro-grid/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Wrap hero or section content — the grid scrolls behind a radial mask while children stay in the overlay slot. Grid lines carry a faint ember tint by default; pass `isBrandTinted={false}` for a neutral grid:

```tsx
import { RetroGrid } from '@/components/animated/retro-grid';

export function Example() {
  return (
    <RetroGrid className="min-h-[400px] rounded-xl">
      <div className="flex min-h-[400px] flex-col items-center justify-center p-8">
        <h2 className="text-2xl font-semibold text-foreground">Your headline</h2>
      </div>
    </RetroGrid>
  );
}
```

## Accessibility

- The grid layer is decorative (`aria-hidden`) and uses `pointer-events-none` — overlay content remains fully interactive.
- Under `prefers-reduced-motion`, the grid plane is static with no scroll keyframes.
- The scroll loop pauses automatically while the component is scrolled offscreen, resuming when it re-enters the viewport.
- Pair with sufficient contrast on overlay text — the radial mask fades lines toward the edges.
