## Installation

Copy the component from `packages/ui/src/components/animated/moving-border/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

Wrap any card or panel, or use the button convenience export:

```tsx
import { MovingBorder, MovingBorderButton } from '@/components/animated/moving-border';

export function Example() {
  return (
    <div className="flex flex-col gap-4">
      <MovingBorder duration={6} borderRadius={12} className="rounded-xl">
        <div className="rounded-xl border border-border bg-card p-6">
          <h3 className="text-lg font-semibold">Pro plan</h3>
        </div>
      </MovingBorder>

      <MovingBorderButton>Get started</MovingBorderButton>
    </div>
  );
}
```

## Accessibility

- The traveling highlight is decorative (`aria-hidden`) and does not affect the accessible name of wrapped content.
- Under `prefers-reduced-motion`, a static subtle brand border replaces the traveling highlight.
- `MovingBorderButton` forwards standard button semantics including `disabled` and focus rings.
