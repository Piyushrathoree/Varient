## Installation

Copy the component from `packages/ui/src/components/animated/browser-frame/` into your project.

```bash
bun add clsx tailwind-merge motion lucide-react
```

## Usage

```tsx
import { BrowserFrame } from '@/components/animated/browser-frame';

export function Example() {
  return (
    <BrowserFrame url="https://varient.dev/docs">
      <div className="px-6 py-8 text-center">
        <p className="font-semibold text-foreground">Your preview here</p>
      </div>
    </BrowserFrame>
  );
}
```

## Accessibility

The frame is a presentational mockup — traffic lights and the globe icon in the URL bar are `aria-hidden`. Content passed as `children` retains its own semantics. When `prefers-reduced-motion` is set, or when `isAnimated` is false, the frame renders statically with no transform animation. Only `opacity` and `transform` animate in the default motion path.
