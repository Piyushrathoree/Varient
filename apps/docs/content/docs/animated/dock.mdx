## Installation

Copy the component from `packages/ui/src/components/animated/dock/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { Dock, DockIcon } from '@/components/animated/dock';
import { Home, Settings } from 'lucide-react';

export function Example() {
  return (
    <Dock iconSize={44} magnification={72} distance={140}>
      <DockIcon label="Home">
        <Home className="size-5" />
      </DockIcon>
      <DockIcon label="Settings">
        <Settings className="size-5" />
      </DockIcon>
    </Dock>
  );
}
```

## Accessibility

- The dock root uses `role="toolbar"`.
- Each `DockIcon` sets `aria-label` from `label` and exposes a native `title` tooltip.
- Magnification is disabled under `prefers-reduced-motion` — icons render at a fixed `iconSize` with no pointer tracking.
