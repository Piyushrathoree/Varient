## Installation

Copy the component from `packages/ui/src/components/animated/marquee/` into your project.

```bash
bun add motion clsx tailwind-merge
```

## Usage

```tsx
import { Marquee } from '@/components/animated/marquee';

export function Example() {
  return (
    <Marquee gap={12} speed={20}>
      <span>Item one</span>
      <span>Item two</span>
      <span>Item three</span>
    </Marquee>
  );
}
```

## Accessibility

- Duplicate content in the loop is marked `aria-hidden` so screen readers do not hear the same items twice.
- The track sets `aria-live="off"` to avoid announcing every scroll tick.
- Under `prefers-reduced-motion`, the marquee renders a static row with no animation and no perpetual motion.
